Project Overview

This project aims to provide a web-based image classification tool leveraging deep learning. Users can upload images or provide URLs, and our model will classify the images into categories such as 'angry', 'crying', 'fighting_dog', 'happy', 'male', 'female', 'running_dog', and 'sitting_dog'.

Features

  • User registration and email verification
  • Password reset functionality
  • Image classification with top-3 predictions
  • User profile management
  • Search history with filters and pagination
  • Secure authentication and session management
  • Responsive and user-friendly interface

Technologies Used

  • Flask: Web framework for Python
  • MySQL: Database management system
  • TensorFlow: Machine learning framework for image classification
  • Flask-Mail: Library for sending emails
  • Flask-Login: User session management
  • Bootstrap: Front-end component library for responsive design

How It Works

  1. Users upload an image or provide an image URL.
  2. The image is preprocessed to match the input format required by the CNN model.
  3. The preprocessed image is fed into the model, which predicts the class probabilities.
  4. The top three predictions are displayed to the user along with their probabilities.
